# Cursor Configuration

Esta pasta contém configurações e padrões específicos do Cursor IDE.

## Estrutura

-  **`standards/`** - Padrões NEØ de desenvolvimento
  -  `README.md` - Visão geral dos padrões
  -  `markdown.rules.md` - Regras de formatação Markdown
  -  `ai.rules.md` - Regras para uso de IA/Cursor
  -  `readme.signature.md` - Assinatura padrão para READMEs

## Uso

O Cursor lê automaticamente o `.cursorrules` na raiz do projeto, que referencia os padrões nesta pasta.
